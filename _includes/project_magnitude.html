Magnitude

Project Summary
An earthquake notification web app. Completed in 2 days. Technologies used: Sinatra, BCrypt, Sqlite3, Javascript, Google Material Design, API calls to Trello & USGS.gov, Git

Usage

Magnitude notifies you when an earthquake occurs. Using Google Maps API, the user selects a point on the map and enters a radius in kms. The user also selects a richter scale and enters their cell phone number.

Using Trello API, the system sends the user a verification code to their cell phone. Once the user enters the correct verification code, the user has successfully subscribed.

Using the USGS.gov API, the system pulls earthquake data periodically and checks for data that fits the user's preferences. If there is a match, the system sends a text to the user.

Duration

Completed the MVP in three days.

Responsibilities

  • Setting up Sinatra, routing, controllers & models.
  • Setting up user authentication model using BCrypt
  • Setting up SQLite 3
  • Site Design using jQuery & Google Material Design
  • Preparing Ajax calls
  • Coding API Calls to USGS.gov & Trello
