Classic Connect 4 with Star Wars Theme written in JS

Project Summary
Classic Connect 4 game written in Javascript. Completed in a day. Technologies used: Javascript MVC Model, Jquery

Usage

It is a classic Connect 4 Game. Each player has 10 seconds time limit. If exceeded, computer automatically plays on your behalf.

Duration

Completed over the weekend with a team of three.

Responsibilities

My role in the project included the following.

  • Creating HTML & CSS based checker board.
  • Figuring out necessary algorithms and implementing MVC structure using Pure Javascript
  • Designing animations using Jquery
