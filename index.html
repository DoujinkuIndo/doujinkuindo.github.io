Link to table :

This tool searches for a gene within ortholog databases, and creates a table of genes named as orthologs to it according to each database. Table rows contain putative orthologs, labeled by Uniprot Accession, Uniprot Gene Name, and number of databases where that gene was called as an ortholog to the query gene. Table column headers contain the database name and the ortholog group ID from that database.

Tables are currently available for ~20,000 human protein coding genes from the 2015 release of the UNIPROT reviewed human proteome ("UP000005640").

The Ortholog Database Linker pulls ortholog groups from:
  • KEGG Orthology (KO) release 74
  • Gene Trees - Ensembl Genome Browser 79
  • OMA groups 2014 release
  • TreeFam release 9
  • OrthoDB version 8
  • EggNOG version 4
  • HOGENOM release 06
  • HOVERGEN release 49
  • COG 2014 version
  • PhylomeDB version 4 - Human Phylome
