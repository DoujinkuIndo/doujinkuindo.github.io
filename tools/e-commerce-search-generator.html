    {% capture about %} ## Background - Why Write Another Keyword Tool For E-Commerce? With all the E-Commerce Keyword Tools out there, why did I bother writing and publishing another one? First and foremost - I like programming and creating things. Secondly, though, this E-Commerce keyword generator does something that I haven't been able to find on any of the other keyword tools - a link to an E-Commerce search results page for that keyword. What's so important about the search results? While keyword suggestions are nice (and an important part of this tool!) I wanted to see what the competition is for each of these keywords with one simple click. And what better place to see the competition than on an actual E-commerce site itself! I plan on using this mostly for my T-Shirt / Print-On-Demand business, but it can also be useful if you sell online in general. ## Technical Stuff This is all written in the browser in JavaScript and uses JQuery. You can view the source either by simply [viewing source on your browser](view-source:https://hendrixjoseph.github.io/tools/e-commerce-keyword-search-generator/), or you can go to [the source hosted on GitHub](https://github.com/hendrixjoseph/hendrixjoseph.github.io/blob/master/tools/e-commerce-search-generator.html). It uses a JSONP callback to an E-Commerce API, which I learned from [this post on Stackoverflow](https://stackoverflow.com/questions/31544085/using-amazons-affiliate-api-for-autocompletion). {% endcapture %} {{ about | markdownify }}
