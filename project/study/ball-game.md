#### Tech stack
Unity3d, C#

#### Project description
Game-ball is a simple arcade game where player control the ball and reach the goal to go further and further.
Game has a timer which goes back and if it will be 0 seconds player will lose. Each coin give bonus time. Aim is to reach final destination on each level.
