#### Tech stack
Android, jUnit, espresso, integration

#### Project description
Android prototype application for hotels & guests.
Make application for remote reception. Application will provide services to many clients and to many hotels. SQLite database on client side will use as cache and contain hotel: information, map, services, room types, restaurant list and etc. Communication guest-hotel will be only for authorised guests by code which possible to receive on reception.
