##### [안드로이드 스튜디오] 기초 함수(1)

```
public void onButton1Clicked(View v) {
        Toast.makeText(getApplicationContext(), "버튼이 눌렸습니다!" , Toast.LENGTH_LONG).show();
    }
```

: 버튼을 클릭했을 때 하단에 "버튼이 눌렸습니다!"라는 문구가 1~3초 등장했다가 사라진다.

```
public void onButton2Clicked(View v) {

    }
```
