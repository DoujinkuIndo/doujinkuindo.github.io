### Python

오랫만에 Python 알고리즘 문제를 풀었다. 어찌어찌 풀고선 다른 사람의 깔끔한 코드를 보니 좌절스..ㅠㅠ 내가 20줄로 푼 문제를 다른 사람들은 단 5줄로 풀었다. 파이썬의 장점을 살려 더욱 깔끔한 코드를 작성하기 위해 노력해야 겠다.

**range**

```python
range(start, stop, step)
x = range(3, 6)

for n in x:
  print(n)

// 3
// 4
// 5
```

**in**

```python
fruits = ["apple", "banana", "cherry"]
if "banana" in fruits:
  print("yes")

// yes
```
