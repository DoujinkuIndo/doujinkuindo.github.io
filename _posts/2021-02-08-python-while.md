## 1. while 사용이 필요한 경우

코드를 작성하다보면, 같은 작업을 반복해야 할 때가 있습니다.

그럴때마다 코드를 복사해서 계속 쓴다면 가독성도 떨어지고,

파일 크기도 늘어나 **비효율적인 방법**이겠죠.

![alt-text](https://github.com/alfonso-john2021/alfonso-john2021.github.io/blob/master/_posts/image/2021-02-08-16-25-06.png?raw=true)

이럴때 바로 while ~~또는 **for**문~~ 을 사용하면 좋습니다.

## 2. 활용은 어떻게

간단합니다. **아래는 ↓ 예시**입니다.

~~~python
#기본적인 양식
while 조건:
    실행될 코드 # 조건이 만족될때 작동한다.

# 계속 반복하고 싶다면?
while True:
    실행될 코드 #무한반복 된다.
~~~

여기서 짚고 넘어가야 할 점이 하나 더 있습니다.

1번째 예시는 상관 없지만, 2번째는 무한하게 반복됩니다.

따라서 `break` 등을 사용하여 끝내야 합니다.

그 이유는 바로 현 시대 컴퓨터에 있습니다.

바로 성능이 너무 좋다는 거죠. (2번째)예시를 실행하게 된다면

1초에 수십만번씩 반복을 하게되고, 그 결과는.. ~~컴퓨터가 과부하 걸린다~~

따라서 `import time`과 `time.sleep(초)`들을 이용해 한바퀴 돌때마다 일정 시간

쉬어주면 그러한 걱정을 할 필요가 없습니다.

## 3. 자주 찾게되는 질문들

Q : 어떤 것들이 가능한가요?

A : `while` 반복문에서는 `if`문이나 `for`문등을 중첩해서 사용할수 있습니다.

반복문의 조건에서는 사칙연산자를 제외한(비교가 가능한) 나머지 연산자들을 사용할수

있고, 1이나 0, True나 False도 사용할 수 있습니다.

Q : 사용시 조심해야 할것은 어떤것들이 있나요?

A : 조건을 1이나 True 로 둔 경우 무한하게 반복되므로 `break` 나 `continue`같은 코드들을

중간에 삽입하여서, 반복이 끝날수 있게 짜는것이 **중요합니다.**

**이러한 코드를 만들었다고 해봅시다.**

~~~python
코드 1

코드 2

while True:
    코드 3
    if 조건:
        break
코드 4
~~~

파이썬은 코드 실행의 순서가 위에서부터 진행되므로 `while`문을 배우지 않았다면

실행 순서가 1->2->3->4 라고 생각하기 쉽지만

조건이 만족하지 않는다면, 1->2->3->3->3 ▪ ▪ ▪ ▪ ▪

조건이 중간에 만족한다면, 1->2->3-#조건이 만족할때>4

이러한 구조로 진행되게 됩니다.

때에 따라서는 반복문 아래의 코드가 영영 실행되지 않을수도 있으니

코드의 실행 순서를 생각하면서 작성하도록 합시다.

## 4. 마치며

반복문을 잘못 만들면 엉뚱한 코드가 실행될수도 있으므로

그점 유의하며 오늘의 ~~짧은~~예시와 함께 마칩니다.

~~~python
while 조건:
    #if와 함께 사용
    if 조건2:
        실행 코드
    실행 코드
~~~
