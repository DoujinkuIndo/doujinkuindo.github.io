# 접기 펼치기

HTML5에서부터 추가된 ```<details>``` 태그를 이용하면 마크다운에서도 접기/펼치기를 할 수 있습니다.

```
<details>
<summary>접기/펼치기</summary>

...
</details>
```

...
</details>

# 접기 펼치기 화살표 달기
```div markdown="1"```은 jekyll에서 html사이에 markdown을 인식하기 위한 코드입니다.

```
<details>
<summary>코드</summary>
<div markdown="1">

...
</details>
```

<details>
<summary>코드</summary>
<div markdown="1">

...
</details>
