- Classification(분류)
  : 미리 정의된, 가능성이 있는 여러 클래스 레이블 중 하나를 예측하는것.
    맞다, 아니다 등의 이진 분류 문제 또는 사과다 바나나다 포도다 등의 2가지 이상으로 분류하는 다중 분류 문제.

- Regression(회귀)
  : 연속적인 숫자(실수)를 예측하는 것
    회귀는 어떤 데이터들의 Feature를 기준으로, 연속된 값(그래프)을 예측하는 문제로 주로 어떤 패턴이나 트렌드, 경향을 예측할 때 사용.

- Feature
  : 머신러닝은 어떤 데이터를 분류하거나, 값을 예측(회귀)하는 것. 
    이렇게 데이터의 값을 잘 예측하기 위한 데이터의 특징들을 머신러닝/딥러닝에서는 "Feature"라고 부르며, 
    지도, 비지도, 강화학습 모두 적절한 feature를 잘 정의하는 것이 핵심임.



1. ML 지도학습
  : 지도학습(Supervised Learning)
    지도 학습은 말 그대로 정답이 있는 데이터를 활용해 데이터를 학습시키며,
    입력 값(X data)이 주어지면 입력값에 대한 Label(Y data)를 주어 학습시키며 대표적으로 분류, 회귀 문제가 있음.

    1) K-NN Regression(K-최근접 이웃 회귀)

    2) 선형회귀

    3) 다중회귀
        - 릿지, 랏쏘

    4) 로지스틱 회귀

    5) 확률적 경사하강법

    6) 결정트리

    7) 교차검증

    8) 트리의 앙상블

2. ML 비지도학습
   : 지도 학습과는 달리 정답 라벨이 없는 데이터를 비슷한 특징끼리 군집화 하여 새로운 데이터에 대한 결과를 예측하는 방법. 
     라벨링 되어있지 않은 데이터로부터 패턴이나 형태를 찾아야 하기 때문에 지도학습보다는 조금 더 난이도가 있음. 
     실제로 지도 학습에서 적절한 피처를 찾아내기 위한 전처리 방법으로 비지도 학습을 이용하기도 함.
