# SHELL

给directory目录添加root权限

```shell
chmod -R 777 directory/
```



如何在任意目录执行某一个可执行文件:方法即将可执行文件的目录添加到PATH环境变量中去，这样就可以任意目录执行了。

```shell
export PATH=/path/to/your/demo:/$PATH
```



查看当前目录下每个文件及文件夹所占内存

```shell
du -ah --max-depth=1   # 查看每个所占内存并列出来，比较慢
du -sh  # 只显示所占的总内存，不列出每个详细，比较快
```
