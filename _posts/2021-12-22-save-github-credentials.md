Cloning a repo from Github can be a pain when you are behind a corporate proxy and forced to use HTTPS instead of SSH having to type your [credentials](https://help.github.com/en/github/authenticating-to-github/creating-a-personal-access-token-for-the-command-line) on every push.
[Caching the credentials](https://stackoverflow.com/a/28562712/2235622)  can soften this pain.
