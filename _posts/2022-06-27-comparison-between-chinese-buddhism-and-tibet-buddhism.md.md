title: 略比较汉传佛教与藏传佛教
---

## 相同、相近之处
* **分类相同**：都是大乘佛教，同属北传佛教。南传上座部为声闻乘，主要在东南亚，所以也称南传。
* **目标相同**：都是以成佛为目标的，声闻乘的目标是涅槃（即阿罗汉位、或无佛、佛法时的辟支佛），而成佛需要行菩萨道，行菩萨道则需要断烦恼障证我空，以及断所知障证法空。
* **净土法门**：作为末法时期的特殊接引方便，两者都特别推崇净土法门，并鼓励信众求生净土，并且都有多位高僧大德发愿往生，以做示范。
* **托钵**：基本上都不托钵了，汉地个别寺院还会。但是东南亚的上座部至今盛行托钵。

## 不同之处

| 比较项 | 汉传佛教 | 藏传佛教 |
| ----- | -------- | ------- |
| 年代 | 早于汉明帝（公元28年）开始，唐宋达到顶峰 | 前宏期（唐朝七世纪始），中间经历朗达玛灭佛，后宏期（宋朝，十世纪） |
| 印度传承 | 大乘佛教初期至密教初期，因此部派佛教的《四阿含》比南传的三藏经典还全面 | 主要是密教时期，部派佛教时期的经典极少，但通过论典学习部派佛教的理论，如《俱舍论》 |
| 地理 | 内地为主，后传至台湾地区、朝鲜、日本、越南等 | 西藏、蒙古（内外）、尼泊尔、不丹、锡金等 |
| 经典 | 包含经、律、论的各种大藏经 | 《甘珠尔》、《丹珠尔》 |
| 派别 | 有八大二小之说，八大为：天台宗、三论宗、律宗、净土宗、法相宗、禅宗、华严宗、密宗。二小为：俱舍宗、成实宗 | 四大主流派别：宁玛派（红教）、噶举派（白教）、萨迦派（花教）、格鲁派（黄教） |
| 显密 | 以显教为绝对主力，但已融入很多原本属于密宗的咒语和仪轨。有过唐密，或称杂密，中断时间较长，后从日本回流至国内。 | 有显教，但都以密教为最高的法，不存在没有密教的教派 |
| 教学体系 | 汉传因为历史的原因，净土和禅宗非常流行，导致了大众对于教理不太深入和系统学习。但是汉传佛教本身的教理体系还是有的，无论是天台，华严还是唯识，都是有非常系统的教理和修学次第的。 | 藏传佛教非常重视次第的教理体系，五部大论的学习，而且有很多对应的学位。特别重视讲法，所以藏传的很多大德都会努力地讲解经典。而且有很多方便法门，照顾教育程度不高的人，如转经筒，经旗，特别重视咒语的诵持等等。 |
| 三无漏学 | 强调戒定慧，净土则强调信愿行，唯识则强调闻思修 | 强调闻思修为主 | 
| 戒律 | 汉传大德的介绍通常都会介绍受具足戒和戒腊 | 相对戒律更强调传承，且女性出家人一直未有戒律传承，已有大德们在呼吁完善戒律传承[1] | 
| 本土影响 | 早期多有借助道家的“无”去解释般若经典，后更有释道儒三教合流之说 | 有不少人认为密宗受到印度教和苯教影响 | 
| 皈依| 汉传佛教是以三皈依为主的，且非常强调“依法不依人”，强调对经典的重视和依止。 | 需皈依上师，且把皈依上师作为第一位，对于上师非常重视，要求“视师如佛”。但也有藏传著名人士指出初学者不该如此。|
| 政教合一 | 无，有过国师级别的高僧辅佐帝王 | 藏传的历史特色之一 |
| 转世 | 无，虽有高僧是某菩萨的化现，但并未体系化 | 藏传的特色之一，含出家人和在家人（瑜伽士） |
| 瑜伽士 | 有，汉传佛教里面虽然也有居士大德，但弘法的主要贡献还是以出家人为绝对主力。 | 藏传有很多高僧大德其实是在家人，但是因为有些人或精通教理，或大力弘扬佛教，在社会上有很高的地位。但这些人依旧可以娶妻生子，喝酒吃肉，过在家人的生活。同时这类瑜伽士也有转世的。 |
| 辩经 | 无 | 藏传特色之一，是一种非常有趣的，可以相互探讨教法的方式。 |
| 仪轨 | 汉传佛教因为受到禅宗和净土的影响，以简洁为主。但也有很多法会，也会使用到很多物品和庄严法会。|藏传的修行特别重视坛城（曼陀罗）的布置，特别重视仪轨，特别重视荟供。|
| 饮食 | 但汉传佛教在梁武帝颁布诏书禁止吃肉之后，就只有素食了。 | 西藏的僧人也是会吃肉的，尤其是以前蔬菜较少时，现在很多藏传大德也推崇素食 |
