## 1. Horizontal Navigation
### Learning Objectives
1. Remove the default styles of an unordered list.
2. Modify links to look like navigation buttons.
3. Create a horizontal navigation bar using Flexbox.

See: [Basic Flexbox Navigation](https://codepen.io/browsertherapy/pen/BaBXjVL)

## 2. Pseudo-selectors and HSL
### Learning Objectives
1. Change the presentation of a navigation link when the user hovers over it.
2. Change the presentation of a navigation link when the user clicks on it.
3. Use HSL to ensure navigation text is readable for the above link states.
