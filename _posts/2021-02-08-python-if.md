## 1. if문 사용이 필요한 경우

코드를 작성하다 보면 조건에 따라 다른 작업을 실행시켜야 할 때가 있습니다.

그럴때 바로 if문을 사용합니다.

~~~python
a = 200
b = 200

if a == b:
    print(a+b)
~~~

위의 코드는 a 와 b 의 값을 비교하여 두 값이 같을 경우,

a 와 b의 합을 출력합니다.

결과는 **200** 이 나오게 됩니다.

걱정하지 않아도 됩니다. **지금 해석할수 없는건 당연합니다.**

## 2. 들여쓰기&문법

전편들을 보았다면 아마

~~~python
a = 200
b = 'hello'

print(a)
print(b)
~~~

이정도 코드는 손쉽게 작성할 수 있을 겁니다.

그렇다면 이제 들여쓰기에 대해 배워봅시다.

~~~python
a = 200
b = 200

if a == b:
    print(a+b)
~~~

아래는 **코드설명** 입니다.

a = 200, b = 200
a와 b에 각각 200을 대입해 줍니다.  
>**추가하자면**

수학에서의 = 기호는 기호 왼쪽의 값과 오른쪽의 값이 같다는 뜻입니다.

그러나 Python 에서는 왼쪽에 오른쪽의 값을 대입하는 기호로 사용됩니다.

또한 나중에 [여기](#22-비교-연산자-&-조건)에서 나오겠지만, 기호 2개를 이어붙이면

"=="이러한 모양으로, 원래의 뜻인 서로 같다는 의미이며 비교 연산자로 사용됩니다.

본론으로 돌아와서, 들여쓰기는 띄어스기 4칸을 대신합니다. `Tab` 키를 사용하여,

들여쓰기를 할 수 있습니다.

**아래**는 if문의 형식입니다.

~~~python
if 조건문:
    실행될 코드 #이 줄에 들여쓰기가 적용되었습니다.
else:
    실행될 코드 # if문의 조건과 틀린경우, 실행됩니다. 
    #마찬가지로 들여쓰기가 적용되어 있습니다. 
~~~

***

### 2.1. 문법

**if문 활용 예시↓** 들여쓰기 된 곳을 찾아봅시다.

~~~python
a = "python"
b = "python"

if a == b:
    print(a)
    print(b)
~~~

1~2 라인: 비교를 위해 변수들을 지정

4 라인: if문 사용, 조건 확인

5~6 라인: if문의 결과가 True 일때 a와 b를 출력

..

..

찾으셨나요? 바로 **5~6**라인이 들여쓰기가 적용되어 있습니다.

특징은 if문 또는 나중에 배울 for, while 등에서 사용되며,

" : " (콜론) 다음 줄부터 적용됩니다.

vscode의 경우 Python 이 설치되어 있다면

.py인 파일에서 알아서 들여쓰기를 적용해 줍니다.

그러나 때에 따라 적용되지 않을때도 있으니 `Tab` 키를 사용해

들여쓰기를 넣어 줍시다.

### 3. 사칙 연산자 & 관계 연산자 & 논리 연산자

#### 3.1. 사칙 연산자

아래의 코드에서

a와 b가 모두 숫자일때

~~~python
a + b
a - b
a * b
a / b
a ** b
~~~

첫번째 부터

더하기, 빼기, 곱하기, 나누기, 거듭제곱입니다.

#### 3.2. 관계 연산자

python 에서는 True는 1을, False는 0을 가집니다.

다음은 관계 연산자 입니다.

##### 3.2.1. 관계 연산자 표

|연산자|의미|
|:-----:|:---|
| a == b |A가 B가 같으면 참|
| a != b |A가 B가 다르면 참|
| a < b |A가 B보다 작으면 참|
| a <= b |A가 B보다 작거나 같으면 참|
| a > b |A가 B보다 크면 참|
| a => b |A가 B보다 크거나 같으면 참|

어떤가요? 상당히 직관적인 형태입니다.

#### 3.3. 논리 연산자

|연산자|의미|
|:-----:|:---|
| a and b |A, B 모두가 참이면 참|
| a or b|A, B 중 하나 이상이 참이면 참|
|  not a|A 논리값의 반대|

위의 세가지 연산자 모두 조건문, 반복문 등에서 사용할수 있습니다.

## 4. 하나더

if문은 아래와 같은 형식으로 이어나갈수 있습니다.

if ~ elif ~ else

당연히 if문 안에 if문 즉 **중첩또한 가능합니다.**

간단한 if문 중첩 예제와 함께 여기서 마치겠습니다.

~~~python
if 조건:
    실행코드
    if 조건2:
        실행코드
    else:
        실행코드
elif 조건:
    if 조건:
        실행코드
else:
    실행코드
~~~
