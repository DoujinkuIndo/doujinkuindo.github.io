### Cách viết bài
Blog này được lập ra trên github page, mọi bài viết sẽ được viết với cú pháp markdown [Xem Markdown cheatsheet](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet)


![First Post](/images/first-post3.png "First Post")

> Để thêm bài viết mới, bạn có thể tạo trực tiếp trên browser trong GitHub.com cũng được! Bấm vào button "Add file" trong thư mục `/_posts/` . Hãy chắc chắn là bài post của bạn đúng format, nếu không nó sẽ không được hiện ra. [front-matter](http://jekyllrb.com/docs/frontmatter/)

Tên bài viết có định dạng "yyyy-MM-dd-tiêu-đề-bài-viết.md"

Hãy tạo pull request như bình thường để có người review trước khi được public nha nha nha.<br/>
